# Fluxinvoice

A minimal, beautiful invoicing application designed for modern teams, built with Next.js 15, Tailwind CSS, and Typescript.

## Getting Started

1. Install dependencies
```bash
pnpm install
```

2. Run the development server
```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
